A tiny little contribution to AngularJS and ngTranslate community.

Based on the awesome Angular-translate

"Multilingual development made easy"

Translators can't handle JSON files that developers give them. They need a tool.

en.json sample 
  {
    "header": {
      "notification": {
        "TITLE": "Notifications",
        "NO_NEWS": "You have no notifications",
        "SEE_ALL": "See all the notifications"
      }
    },
    "aside": {
      "nav": {
        "LOGOUT": "Logout"
      }
    }
  }
